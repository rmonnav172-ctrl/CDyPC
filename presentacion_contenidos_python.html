Instrucciones y Secuenciación

¿Qué es una instrucción?

Una instrucción es cada una de las órdenes que damos al ordenador para que ejecute una operación. Es un conjunto de datos insertados en una secuencia específica que el procesador interpreta y ejecuta.

Ejemplo básico de instrucción
print("¡Hola Mundo!")

Esta simple línea le dice al ordenador: "muestra el texto '¡Hola Mundo!' en la pantalla"

Secuenciación

La secuenciación es la organización correcta de las acciones. Las instrucciones se ejecutan una tras otra, en orden secuencial. El orden importa.

Analogía: Hacer un bocadillo
  1. Coger dos rebanadas de pan
  2. Untar mantequilla en una rebanada
  3. Poner jamón sobre la mantequilla
  4. Cubrir con la segunda rebanada

¿Qué pasaría si cambiamos el orden?

# Secuencia de instrucciones print("Hola, ¿cómo estás?") nombre = input("¿Cómo te llamas? ") print("Hola", nombre, "encantado de saludarte")

Algoritmos y Diagramas de Flujo

¿Qué es un algoritmo?

Un algoritmo es un conjunto de instrucciones o reglas definidas, ordenadas y finitas que permite solucionar un problema, realizar un cómputo o llevar a cabo otras tareas.

Algoritmo: Freír un huevo
  1. Colocar una sartén sobre la vitrocerámica
  2. Encender la vitro a media potencia
  3. Verter aceite en la sartén
  4. Romper un huevo y verterlo en la sartén
  5. Añadir una pizca de sal sobre la yema
  6. Esperar 2 minutos
  7. Apagar el fuego
Algoritmo: Área de triángulo
  1. Medir la base
  2. Medir la altura
  3. Multiplicar base por altura
  4. Dividir el resultado por 2
base = 7 altura = 10 area = (base * altura) / 2 print("El área es:", area)
Diagramas de flujo

Los diagramas de flujo son representaciones gráficas que ilustran el recorrido de las acciones. Son como mapas que nos guían a través de diferentes caminos según las decisiones que tomemos.

Tipos de Datos en Python

Cuando programamos, trabajamos con diferentes tipos de información. Python clasifica estos datos en categorías específicas.
Los cuatro tipos principales
Tipo Descripción Ejemplo
int (Enteros) Números sin decimales 5, -3, 1000
float (Flotantes) Números con decimales 3.14, -2.5, 0.0
str (Cadenas) Texto entre comillas "Hola", 'Python'

# Diferentes tipos de datos edad = 16 # int altura = 1.75 # float nombre = "Ana" # str es_estudiante = True # bool # Verificar el tipo print(type(edad)) # print(type(altura)) #
¿Por qué son importantes los tipos?

Los tipos determinan qué operaciones podemos realizar. No puedes sumar un número con una palabra, pero sí puedes concatenar dos palabras.

Variables y Constantes

¿Qué son las variables?

Las variables son espacios en la memoria del ordenador que utilizamos para almacenar datos. Son como "cajones con etiquetas" donde guardamos información que podemos usar más tarde.

Creación y uso de variables
# Crear variables edad = 25 nombre = "Juan" altura = 1.75 es_estudiante = True # Usar las variables print("Me llamo", nombre) print("Tengo", edad, "años") # Modificar variables edad = edad + 1 print("Ahora tengo", edad, "años")
Variables
  • Pueden cambiar de valor
  • Se escriben en minúsculas
  • Nombres descriptivos
  • Ejemplo: edad, nombre
Constantes
  • No cambian de valor
  • Se escriben en MAYÚSCULAS
  • Valores fijos del programa
  • Ejemplo: PI, GRAVEDAD
# Ejemplos de constantes PI = 3.14159 GRAVEDAD = 9.81 VELOCIDAD_LUZ = 299792458

Estructuras Condicionales

¿Qué son las estructuras condicionales?

Las estructuras condicionales permiten que nuestro programa tome decisiones. Según se cumplan o no ciertas condiciones, se ejecutarán diferentes bloques de código.

Estructura if-elif-else
edad = int(input("¿Cuántos años tienes? ")) if edad < 13: print("Eres un niño/a") elif edad < 18: print("Eres un/a adolescente") elif edad < 65: print("Eres un/a adulto/a") else: print("Eres un/a adulto/a mayor")
Palabras clave
  • if: "si" se cumple la condición
  • elif: "sino, si" se cumple otra condición
  • else: "sino" (cuando no se cumple ninguna)
Reglas importantes
  • Usar : después de la condición
  • Indentar el código interno
  • Solo se ejecuta un bloque
  • else es opcional
Ejemplo: Sistema de notas
nota = float(input("Introduce tu nota: ")) if nota >= 9: print("Sobresaliente") elif nota >= 7: print("Notable") elif nota >= 5: print("Aprobado") else: print("Suspenso")

Estructuras Repetitivas (Bucles)

¿Por qué usar bucles?

Los bucles nos permiten repetir código sin escribirlo múltiples veces. Son fundamentales para automatizar tareas repetitivas.

Bucle FOR - Repeticiones conocidas

Usamos for cuando sabemos cuántas veces queremos repetir algo

# Contar del 0 al 4 for i in range(5): print(i) # 0, 1, 2, 3, 4 # Contar desde 2 hasta 6 for i in range(2, 7): print(i) # 2, 3, 4, 5, 6 # Contar de 2 en 2 for i in range(0, 11, 2): print(i) # 0, 2, 4, 6, 8, 10 # Cuenta atrás for i in range(5, 0, -1): print(f"{i}...") print("¡Despegue!")
Bucle WHILE - Repetir hasta que...

Usamos while cuando repetimos hasta que se cumpla una condición

contador = 0 while contador < 5: print(f"Vuelta número {contador}") contador = contador + 1 # Juego de adivinanza numero_secreto = 7 adivinado = False while not adivinado: intento = int(input("Adivina el número (1-10): ")) if intento == numero_secreto: print("¡Correcto!") adivinado = True else: print("Inténtalo de nuevo")
Ejemplo: Tabla de multiplicar
numero = int(input("¿Qué tabla quieres ver? ")) print(f"Tabla del {numero}:") for i in range(1, 11): resultado = numero * i print(f"{numero} x {i} = {resultado}")

Listas - Colecciones de datos

¿Qué son las listas?

Las listas son colecciones ordenadas y modificables de elementos. Son como "cajas con compartimentos numerados" donde cada elemento tiene una posición específica (índice).

Crear y manipular listas
# Crear listas frutas = ["manzana", "plátano", "cereza"] numeros = [1, 5, 3, 9, 2] mixta = ["Ana", 16, True, 1.65] # Acceder a elementos (empezamos en 0) print(frutas[0]) # "manzana" print(frutas[2]) # "cereza" print(frutas[-1]) # "cereza" (último elemento) # Modificar elementos frutas[1] = "kiwi" print(frutas) # ["manzana", "kiwi", "cereza"]
Métodos útiles de listas
aficiones = ["leer", "nadar"] # Añadir elementos aficiones.append("programar") # Al final aficiones.insert(1, "correr") # En posición específica # Eliminar elementos aficiones.remove("nadar") # Por valor ultima = aficiones.pop() # Elimina y devuelve el último # Información de la lista print(len(aficiones)) # Longitud print("leer" in aficiones) # ¿Está el elemento?
Recorrer listas
colores = ["rojo", "verde", "azul"] # Método 1: Recorrer elementos directamente for color in colores: print(f"Me gusta el {color}") # Método 2: Recorrer con índices for i in range(len(colores)): print(f"{i}: {colores[i]}") # Método 3: Con enumerate (índice y valor) for i, color in enumerate(colores): print(f"Posición {i}: {color}")

Funciones - Código Reutilizable

¿Qué son las funciones?

Las funciones son bloques de código que realizan una tarea específica y se pueden invocar desde otras partes del programa. Son como "máquinas" que reciben datos, los procesan y pueden devolver un resultado.

Estructura de una función
# Función sin parámetros def saludar(): print("¡Hola a todos!") saludar() # Llamar a la función # Función con parámetros def saludar_persona(nombre): print(f"¡Hola, {nombre}!") saludar_persona("Ana") saludar_persona("Carlos")
Funciones que devuelven valores
def calcular_area_rectangulo(base, altura): area = base * altura return area # Usar la función mi_area = calcular_area_rectangulo(5, 3) print(f"El área es: {mi_area}") # Función para calcular promedio def calcular_promedio(nota1, nota2, nota3): promedio = (nota1 + nota2 + nota3) / 3 return promedio mi_promedio = calcular_promedio(8, 7, 9) print(f"Tu promedio es: {mi_promedio}")
Ventajas de las funciones
  • Reutilizar código
  • Organizar mejor el programa
  • Facilitar el mantenimiento
  • Probar partes por separado
  • Trabajar en equipo
Alcance de variables
  • Local: Dentro de la función
  • Global: Fuera de funciones
  • Variables locales no se ven fuera
  • Evitar variables globales
Ejemplo completo: Calculadora
def sumar(a, b): return a + b def restar(a, b): return a - b def multiplicar(a, b): return a * b def dividir(a, b): if b != 0: return a / b else: return "Error: División por cero" # Usar las funciones resultado = sumar(10, 5) print(f"10 + 5 = {resultado}")
